# Plan 09-03: Heatmap LiveView Component (2D Sharpe Grid + Click-Through)

## Objective
Build an interactive 2D heatmap visualization where users select two optimizer parameters as X/Y axes, cells show Sharpe ratio (color-coded), and clicking a cell navigates to the full results for that combo.

## Requirements Addressed
- OPT-04: Heatmap visualization (X/Y axes = 2 params, cell color = Sharpe, grey = <20 trades)
- UI-12: Optimizer dashboard with param grid config and 2D heatmap

## Wave Execution

### Wave 1: Heatmap JS Hook (Canvas-based)
**Files:**
- `apps/quantedge_web/assets/js/optimizer_heatmap_hook.js`
- `apps/quantedge_web/assets/js/app.js` — register hook

**Details:**
Create a Phoenix LiveView JS hook `OptimizerHeatmap` that renders a 2D grid:

Hook interface:
```javascript
Hooks.OptimizerHeatmap = {
  mounted() {
    this.renderHeatmap(JSON.parse(this.el.dataset.heatmapData));
  },
  updated() {
    this.renderHeatmap(JSON.parse(this.el.dataset.heatmapData));
  },
  renderHeatmap(data) {
    // data shape: {
    //   x_param: "sl_value",
    //   y_param: "strike_offset",
    //   x_values: [20, 25, 30, 35, 40, 45, 50],
    //   y_values: [-2, -1, 0, 1, 2],
    //   cells: [{x: 20, y: 0, sharpe: 1.85, trade_count: 892, pnl: 234567}, ...]
    // }
    //
    // Render:
    // - Canvas element sized to grid dimensions
    // - X-axis labels: x_values
    // - Y-axis labels: y_values
    // - Cell color: diverging colormap (red → yellow → green) based on Sharpe
    //   - Sharpe < 0: red (#e74c3c)
    //   - Sharpe 0-1: yellow (#f39c12)
    //   - Sharpe 1-2: green (#27ae60)
    //   - Sharpe > 2: dark green (#1a7a3a)
    //   - trade_count < 20: grey (#555) with "?" label
    // - Cell text: Sharpe value (2 decimal places)
    // - On cell click: push event "heatmap_cell_click" with {x, y, combo_index}
    // - Tooltip on hover: "SL=25, Offset=0 | Sharpe: 1.85 | PnL: ₹2.3L | Trades: 892"
  }
}
```

Color scale using HSL interpolation for smooth gradients:
- Map Sharpe range [-1, 3] to hue [0° (red) → 120° (green)]
- Saturation: 70%, Lightness: 45%

### Wave 3: Click-Through Detail Modal
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/optimizer_live.ex` — modal rendering

**Details:**
When user clicks a heatmap cell:
1. Find the full `OptimizerResult` for that combo
2. Display in a modal overlay:
   - **Parameter values**: all param values for this combo

# Plan 06-01: Phoenix Umbrella App Scaffold

## Objective
Create the Elixir/Phoenix umbrella project structure within the existing repo. This establishes the Elixir side of the platform — `quantedge_web` (Phoenix), `quantedge` (business logic), and wires up Rustler to compile the existing NIF crate.

## Requirements Covered
- NIF-01 (partial): Elixir-side module stub for `QuantEdge.NIF`
- STR-01 (partial): Phoenix/Ecto project scaffold with Postgres config

## Context
- Currently the repo is Rust-only (`Cargo.toml` workspace at root)
- The NIF crate (`apps/quantedge_core/crates/nif`) already exists with a stub `lib.rs`
- Phoenix umbrella apps go alongside the Rust workspace under `apps/`
- Stack spec: Elixir 1.17+, Phoenix 1.7+, LiveView 1.0+, Oban 2.17+

## Implementation Steps

### Step 1: Generate Phoenix umbrella app
```bash
# Create umbrella structure in apps/
cd apps/
mix phx.new quantedge_web --umbrella --live --database postgres --no-install
```
This creates:
- `apps/quantedge/` — business logic app (Ecto schemas, NIF bridge, workers)
- `apps/quantedge_web/` — Phoenix web app (LiveView, routes, templates)

### Step 2: Configure Rustler in `quantedge` app
```elixir
# apps/quantedge/mix.exs — add rustler dependency
defp deps do
  [
    {:rustler, "~> 0.35"},
    {:ecto_sql, "~> 3.11"},
    {:postgrex, ">= 0.0.0"},
    {:oban, "~> 2.17"},
    {:jason, "~> 1.4"},
    {:phoenix_pubsub, "~> 2.1"},
  ]
end

# Rustler config
rustler_crates: [
  quantedge_nif: [
    path: "../quantedge_core/crates/nif",
    mode: if(Mix.env() == :prod, do: :release, else: :debug)
  ]
]
```

### Step 3: Create NIF module stub
```elixir
# apps/quantedge/lib/quantedge/nif.ex
defmodule QuantEdge.NIF do
  use Rustler, otp_app: :quantedge, crate: "quantedge_nif"

  def run_backtest(_strategy_json, _opts_json),       do: :erlang.nif_error(:not_loaded)
  def run_optimizer(_strategy_json, _param_grid),     do: :erlang.nif_error(:not_loaded)
  def run_portfolio(_strategies_json, _opts_json),    do: :erlang.nif_error(:not_loaded)
  def load_data_summary(_symbol, _from, _to),         do: :erlang.nif_error(:not_loaded)
end
```

### Step 4: Configure Postgres in dev/test
```elixir
# config/dev.exs
config :quantedge, QuantEdge.Repo,
  username: "postgres",
  password: "postgres",
  hostname: "localhost",
  database: "quantedge_dev",
  stacktrace: true,
  show_sensitive_data_on_connection_error: true,
  pool_size: 10
```

### Step 5: Configure PubSub
```elixir
# apps/quantedge/lib/quantedge/application.ex
children = [
  QuantEdge.Repo,
  {Phoenix.PubSub, name: QuantEdge.PubSub},
  {Oban, Application.fetch_env!(:quantedge, Oban)},
]
```

### Step 6: Verify compilation
```bash
mix deps.get
mix compile
# NIF should compile via Rustler and load
```

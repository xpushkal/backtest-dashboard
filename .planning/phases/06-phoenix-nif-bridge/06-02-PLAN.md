# Plan 06-02: Rustler NIF Functions with Dirty CPU Scheduling

## Objective
Implement the three core NIF functions (`run_backtest`, `run_optimizer`, `run_portfolio`) plus `load_data_summary` in the existing `quantedge-nif` Rust crate. All NIFs run on `dirty_cpu` scheduler to never block the BEAM.

## Requirements Covered
- NIF-01: run_backtest NIF (dirty CPU, JSON in → JSON out)
- NIF-02: run_optimizer NIF (parallel grid sweep via Rayon)
- NIF-03: run_portfolio NIF (multi-strategy simultaneous)
- NIF-04: NIF call overhead <5ms (empty backtest round-trip)

## Context
- NIF crate exists at `apps/quantedge_core/crates/nif/` with stub `init!`
- Depends on: `quantedge-core` (SimRunner, StrategyConfig), `quantedge-data` (Parquet loading), `quantedge-metrics` (MetricsEngine), `quantedge-greeks` (attribution)
- PRD pattern: JSON string in → deserialize → run → serialize → JSON string out
- All NIFs must use `#[rustler::nif(schedule = "DirtyCpu")]`

### Step 2: Implement run_backtest NIF
```rust
// src/lib.rs
#[rustler::nif(schedule = "DirtyCpu")]
fn run_backtest(strategy_json: String, opts_json: String) -> Result<String, String> {
    // 1. Deserialize strategy config
    let request: BacktestRequest = serde_json::from_str(&opts_json)
        .map_err(|e| format!("Failed to parse request: {}", e))?;

    // 2. Parse strategy config from TOML embedded in JSON
    let config = parse_strategy_config(&request.strategy)?;

    // 3. Load bar data from Parquet
    let bars = load_bars(&request)?;

    // 4. Run simulation
    let result = SimRunner::run(&config, &bars, request.lot_size);

    // 5. Compute metrics
    let trade_records = convert_trades(&result);
    let equity_points = convert_equity(&result, request.capital);
    let metrics = MetricsEngine::compute(
        &trade_records, &equity_points, request.capital,
        parse_date(&request.date_from)?,
        parse_date(&request.date_to)?,
    );

    // 6. Serialize response
    let response = BacktestResponse { ... };
    serde_json::to_string(&response).map_err(|e| e.to_string())
}
```

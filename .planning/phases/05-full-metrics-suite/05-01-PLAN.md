# Plan 05-01: Greeks Engine — Black-Scholes Pricer

## Objective
Implement a full Black-Scholes option pricing engine in the `quantedge-greeks` crate. Compute delta, gamma, theta, vega for both CE and PE options. This engine is the foundation for Greeks PnL attribution (Plan 05-02) and options-specific metrics (Plan 05-04).

## Context
- **Crate**: `apps/quantedge_core/crates/greeks/` — currently empty stub (`lib.rs` has 2 lines)
- **Dependencies**: `statrs` 0.17 (normal distribution CDF/PDF), `ndarray` 0.16, `chrono` 0.4
- **IV surface**: Already available in `quantedge-data::IvSurface` (cubic spline interpolation)
- **Success criterion 2**: Black-Scholes prices match reference within 0.01%
- **Risk-free rate**: 6.5% (India) — hardcoded constant

### 1. `greeks/src/black_scholes.rs` (NEW — ~200 lines)

```rust
// Core Black-Scholes implementation

pub struct BsInputs {
    pub spot: f64,           // Current underlying price
    pub strike: f64,         // Option strike price
    pub tte: f64,            // Time to expiry in years (days/365.25)
    pub iv: f64,             // Implied volatility (decimal, e.g. 0.15 for 15%)
    pub risk_free_rate: f64, // Default: 0.065 (India)
}

pub struct BsOutput {
    pub price: f64,
    pub delta: f64,
    pub gamma: f64,
    pub theta: f64,  // Per-day theta (divide annual by 365.25)
    pub vega: f64,   // Per 1% IV move
}

pub const RISK_FREE_RATE_INDIA: f64 = 0.065;

// Formulas:
// d1 = (ln(S/K) + (r + σ²/2) * T) / (σ * √T)
// d2 = d1 - σ * √T
//
// CE price = S * N(d1) - K * e^(-rT) * N(d2)
// PE price = K * e^(-rT) * N(-d2) - S * N(-d1)
//
// delta_CE = N(d1),          delta_PE = N(d1) - 1
// gamma    = n(d1) / (S * σ * √T)
// theta_CE = -(S * n(d1) * σ) / (2√T) - r * K * e^(-rT) * N(d2)
// theta_PE = -(S * n(d1) * σ) / (2√T) + r * K * e^(-rT) * N(-d2)
// vega     = S * n(d1) * √T / 100   (per 1% IV move)
//
// N(x) = standard normal CDF  (statrs::distribution::Normal)
// n(x) = standard normal PDF
```

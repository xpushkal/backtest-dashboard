# Plan 05-05: Walk-Forward Analysis Engine

## Objective
Implement windowed in-sample/out-of-sample walk-forward analysis. Produces ~12 IS/OOS pairs from 4 years of data, reports IS Sharpe, OOS Sharpe, and degradation ratio per window. This validates strategy robustness against overfitting.

## Context
- **PRD specification**:
  ```
  Total period: 4 years
  Window: 6 months in-sample, 2 months out-of-sample
  Slide: 2 months
  Result: ~12 IS/OOS pairs
  Report: IS Sharpe, OOS Sharpe, degradation ratio (OOS/IS) per window
  ```
- **Success Criterion 4**: 12 windows produced, IS and OOS periods strictly non-overlapping
- **Key insight**: Walk-forward does NOT re-run backtests — it slices existing trade results by date range and recomputes metrics per window. This makes it fast (no re-simulation).

### 1. `metrics/src/walk_forward.rs` (NEW — ~200 lines)

```rust
/// Walk-forward analysis configuration.
pub struct WalkForwardConfig {
    pub is_months: u32,    // In-sample period (default: 6)
    pub oos_months: u32,   // Out-of-sample period (default: 2)
    pub slide_months: u32, // Slide step (default: 2)
}

impl Default for WalkForwardConfig {
    fn default() -> Self {
        Self { is_months: 6, oos_months: 2, slide_months: 2 }
    }
}

/// One walk-forward window result.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WfWindow {
    pub window_id: u32,
    pub is_start: NaiveDate,
    pub is_end: NaiveDate,
    pub oos_start: NaiveDate,
    pub oos_end: NaiveDate,
    pub is_sharpe: f64,
    pub oos_sharpe: f64,
    pub degradation_ratio: f64,  // oos_sharpe / is_sharpe (< 1 = degradation)
    pub is_trades: u32,
    pub oos_trades: u32,
    pub is_pnl: f64,
    pub oos_pnl: f64,
}

/// Walk-forward summary.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WalkForwardResult {
    pub windows: Vec<WfWindow>,
    pub avg_degradation: f64,     // mean of degradation_ratios
    pub pct_positive_oos: f64,    // % windows where OOS Sharpe > 0
    pub robustness_score: f64,    // avg_degradation × pct_positive_oos
}

pub struct WalkForwardEngine;

impl WalkForwardEngine {
    /// Run walk-forward analysis on existing trade results.
    ///
    /// Slices trades and equity by date window, computes per-window Sharpe.
    /// Does NOT re-run simulation — only recomputes metrics per slice.
    pub fn analyze(
        trades: &[TradeRecord],
        equity: &[EquityPoint],
        capital: f64,
        data_start: NaiveDate,
        data_end: NaiveDate,
        config: &WalkForwardConfig,
    ) -> WalkForwardResult { ... }

    /// Generate window date ranges.
    fn generate_windows(
        start: NaiveDate,
        end: NaiveDate,
        config: &WalkForwardConfig,
    ) -> Vec<(NaiveDate, NaiveDate, NaiveDate, NaiveDate)> { ... }

    /// Compute Sharpe for trades within a date range.
    fn sharpe_for_range(
        trades: &[TradeRecord],
        equity: &[EquityPoint],
        start: NaiveDate,
        end: NaiveDate,
    ) -> f64 { ... }
}
```

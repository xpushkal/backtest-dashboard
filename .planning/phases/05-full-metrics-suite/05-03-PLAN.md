# Plan 05-03: Complete Risk Metrics

## Objective
Expand the metrics engine from 20 to 40+ metrics by adding all remaining risk and return metrics: Sortino, Calmar, Omega, VaR, CVaR, Ulcer Index, skewness, kurtosis, recovery factor, consecutive wins/losses, win/loss ratio, gross profit/loss, and drawdown duration.

## Context
- **Existing**: `MetricsResult` has 20 metrics (11 return + 3 risk + 6 trade)
- **PRD metrics to add**: 
  - Return: `win_loss_ratio`, `gross_profit`, `gross_loss`
  - Risk: `sortino_ratio`, `calmar_ratio`, `omega_ratio`, `var_95`, `var_99`, `cvar`, `ulcer_index`, `daily_volatility`, `ann_volatility`, `skewness`, `kurtosis`, `recovery_factor`, `avg_drawdown`, `drawdown_duration_bars`
  - Trade: `max_hold_bars`, `max_consec_wins`, `max_consec_losses`, `reentry_count`, `reentry_win_rate`, `total_slippage`, `total_stt`, `net_cost_ratio`
- **Risk-free rate**: 6.5% (hardcoded, India)
- **Formulas from PRD**:
  ```
  Sortino = (ann_return - 0.065) / downside_deviation
  Calmar  = CAGR / max_drawdown_pct
  VaR 95  = percentile(daily_pnl_series, 5)
  CVaR    = mean(daily_pnl < VaR_95)
  Ulcer   = sqrt(mean(((equity - rolling_max) / rolling_max × 100)²))
  Omega   = sum(returns > threshold) / |sum(returns < threshold)|
  ```

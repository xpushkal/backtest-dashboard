# Plan 05-04: Options-Specific Metrics

## Objective
Add 14 options-specific metrics to the metrics engine: premium capture, theta collected, IV crush analysis, DTE distribution, breakeven range, and max theoretical profit. These are unique to options backtesting and differentiate QuantEdge from generic tools.

## Context
- **PRD options metrics (14)**: `premium_capture_pct`, `total_theta_collected`, `avg_theta_per_day`, `avg_iv_at_entry`, `avg_iv_at_exit`, `iv_crush_pct`, `delta_pnl`, `gamma_pnl`, `theta_pnl`, `vega_pnl`, `avg_net_delta`, `dte_distribution`, `breakeven_range`, `max_profit_theoretical`
- **Greeks attribution** (05-02) already produces per-trade delta/gamma/theta/vega PnL — this plan aggregates them
- **IV data**: Available from bar data (`iv` field in SimBar/CSV)
- **DTE**: Computed from entry date + expiry date (expiry calendar in quantedge-data)

## Files to Create/Modify

### 1. `metrics/src/options_metrics.rs` (NEW — ~200 lines)

```rust
use crate::core_metrics::TradeRecord;

/// Options-specific metrics result.
#[derive(Debug, Clone, Default, Serialize, Deserialize)]
pub struct OptionsMetrics {
    // ── Premium & Theta (3) ──
    pub premium_capture_pct: f64,    // (premium_collected - premium_returned) / premium_collected × 100
    pub total_theta_collected: f64,  // sum of theta_pnl for sell positions
    pub avg_theta_per_day: f64,      // total_theta / total_days_held

    // ── IV Analysis (4) ──
    pub avg_iv_at_entry: f64,
    pub avg_iv_at_exit: f64,
    pub iv_crush_pct: f64,          // (avg_iv_entry - avg_iv_exit) / avg_iv_entry × 100
    pub avg_net_delta: f64,         // avg position delta across all trades

    // ── Distribution (3) ──
    pub dte_distribution: DteDistribution,
    pub breakeven_range: (f64, f64),   // (lower_breakeven, upper_breakeven) in spot terms
    pub max_profit_theoretical: f64,   // for sell: total premium; for buy: unlimited (capped)

    // ── Greeks PnL aggregate (4 — from attribution) ──
    pub total_delta_pnl: f64,
    pub total_gamma_pnl: f64,
    pub total_theta_pnl: f64,
    pub total_vega_pnl: f64,
}

#[derive(Debug, Clone, Default, Serialize, Deserialize)]
pub struct DteDistribution {
    pub avg_dte: f64,
    pub min_dte: f64,
    pub max_dte: f64,
    pub pct_below_3: f64,    // % trades entered with DTE < 3
    pub pct_3_to_7: f64,     // 3-7 DTE
    pub pct_above_7: f64,    // > 7 DTE
}

pub struct OptionsMetricsEngine;

impl OptionsMetricsEngine {
    pub fn compute(trades: &[OptionsTradeRecord]) -> OptionsMetrics { ... }
}
```

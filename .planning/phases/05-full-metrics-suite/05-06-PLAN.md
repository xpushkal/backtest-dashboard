# Plan 05-06: Monte Carlo Simulation Engine

## Objective
Implement trade-sequence-shuffle Monte Carlo simulation that produces percentile equity bands and probability statistics. Uses 1000 simulations to generate confidence intervals for strategy performance.

## Context
- **PRD specification**:
  ```
  N = 1000 simulations
  Method: shuffle trade sequence randomly, recompute equity curve
  Output: 5th / 25th / 50th / 75th / 95th percentile equity curves
  Key stat: probability of positive return at median, P(MDD > X%)
  ```
- **Success Criterion 5**: 1000 sims, percentile bands computed, P(positive return) calculated
- **Performance**: 1000 sims × 1000 trades = 1M equity curve points — must complete < 500ms
- **Method**: Fisher-Yates shuffle on trade PnL sequence, replay to build equity curve, collect final equity and max DD per sim

### 1. `metrics/src/monte_carlo.rs` (NEW — ~250 lines)

```rust
use rand::prelude::*;
use rand::rngs::StdRng;

/// Monte Carlo simulation configuration.
pub struct MonteCarloConfig {
    pub n_simulations: u32,  // default: 1000
    pub seed: Option<u64>,   // for reproducibility in tests
}

impl Default for MonteCarloConfig {
    fn default() -> Self {
        Self { n_simulations: 1000, seed: None }
    }
}

/// Percentile equity at a given point.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct PercentileBand {
    pub trade_index: u32,     // x-axis: trade number
    pub p5: f64,              // 5th percentile equity
    pub p25: f64,
    pub p50: f64,             // median
    pub p75: f64,
    pub p95: f64,
}

/// Monte Carlo simulation results.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct MonteCarloResult {
    /// Percentile bands at each trade index
    pub equity_bands: Vec<PercentileBand>,
    /// Final equity distribution percentiles
    pub final_p5: f64,
    pub final_p25: f64,
    pub final_p50: f64,
    pub final_p75: f64,
    pub final_p95: f64,
    /// Max drawdown distribution
    pub mdd_p50: f64,         // median max DD
    pub mdd_p95: f64,         // 95th percentile max DD (worst case)
    /// Probability statistics
    pub prob_positive_return: f64,    // P(final_equity > capital)
    pub prob_mdd_gt_10pct: f64,       // P(MDD > 10%)
    pub prob_mdd_gt_20pct: f64,       // P(MDD > 20%)
    /// Metadata
    pub n_simulations: u32,
    pub n_trades: u32,
}

pub struct MonteCarloEngine;

impl MonteCarloEngine {
    /// Run Monte Carlo simulation by shuffling trade PnL sequence.
    ///
    /// 1. Extract PnL vector from trades
    /// 2. For each sim: shuffle PnLs, build equity curve, record final equity + max DD
    /// 3. Compute percentile bands across all sims at each trade index
    /// 4. Compute probability statistics from final equity distribution
    pub fn simulate(
        trade_pnls: &[f64],
        capital: f64,
        config: &MonteCarloConfig,
    ) -> MonteCarloResult { ... }

    /// Compute max drawdown from an equity curve.
    fn max_drawdown(equity: &[f64]) -> f64 { ... }

    /// Compute percentile from sorted values.
    fn percentile(sorted: &[f64], p: f64) -> f64 { ... }
}
```

# Plan 07-07: Data Explorer LiveView

## Objective
Data explorer showing loaded symbols, date ranges, bar counts, IV coverage, and data quality indicators.

## Requirements Addressed
- UI-11: Data explorer showing loaded symbols, date ranges, bar counts, IV coverage

## Wave Execution

### Wave 1: Data Explorer LiveView
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/data_explorer_live.ex`
- `apps/quantedge_web/lib/quantedge_web/live/data_explorer_live.html.heex`

**Details:**
Page layout — card per underlying (BANKNIFTY, NIFTY, SENSEX):

Each card shows:
1. **Symbol header** with underlying badge
2. **Date range** covered (first bar date → last bar date)
3. **Bar count** (total 1-min OHLCV bars)
4. **Trading days** count
5. **Expiry count** (weekly + monthly)
6. **IV coverage** percentage (bars with valid IV / total bars)
7. **Data quality indicators:**
   - Missing days (gaps in trading calendar)
   - Zero-volume bars percentage
   - IV=0 bars percentage
8. **Strike coverage:** Min/max strikes available, strike interval

Data source: Call `NIF.load_data_summary(symbol, date_from, date_to)` for each underlying.

### Wave 2: Parquet File Browser
**Files:**
- Same LiveView, additional section

**Details:**
- Expandable section showing raw Parquet files in `Data/parquet/` directory
- File list: filename, size (MB), row count, date range
- Total storage size
- "Refresh" button to re-scan files
- Uses `File.ls` + `File.stat` on the Elixir side (no NIF needed)

## Verification

- [ ] NIF `load_data_summary` is called and results displayed
- [ ] Parquet file browser lists files with sizes

## UAT Criteria

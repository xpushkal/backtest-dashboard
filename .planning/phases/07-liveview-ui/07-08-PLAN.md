# Plan 07-08: Optimizer Dashboard & Portfolio Builder LiveViews

## Objective
Optimizer dashboard with parameter grid configuration and 2D Sharpe heatmap, plus portfolio builder with capital allocation and correlation matrix.

## Requirements Addressed
- UI-12: Optimizer dashboard with param grid config and 2D heatmap
- UI-13: Portfolio builder with capital allocation sliders and correlation matrix

## Wave Execution

### Wave 1: Optimizer Dashboard LiveView
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/optimizer_live.ex`
- `apps/quantedge_web/lib/quantedge_web/live/optimizer_live.html.heex`

**Details:**
Page sections:

1. **Strategy Selector:** Pick strategy to optimize (dropdown)

2. **Parameter Grid Config:**
   - Dynamic form: select parameter → set min/max/step
   - Supported params: sl_value, target_value, entry_time, exit_time, strike_offset, lots
   - Show total combinations count (product of all param ranges)
   - "Start Optimization" button → enqueues OptimizerWorker

3. **Running Optimizations:**
   - Progress bar with combo count (e.g., "245/720 combos completed")
   - Estimated time remaining
   - Live updates via PubSub `optimizer:{run_id}`

4. **Results Section (after completion):**
   - Best parameters summary card (top 5 by Sharpe)
   - Full results data table: Combo #, Params, Sharpe, PnL, Max DD, Win Rate
   - Sortable by any metric column
   - Click row → view full backtest results for that combo

### Wave 2: Optimizer Heatmap
**Files:**
- `apps/quantedge_web/assets/js/hooks/optimizer_heatmap.js` — 2D heatmap hook

**Details:**
- User selects 2 parameters for X and Y axes (dropdowns)
- Cell value: Sharpe ratio (or other selectable metric)
- Color gradient: red (low) → yellow (mid) → green (high)
- Click cell → navigate to full results for that parameter combination
- Axes labeled with parameter values
- Data from `Duck.Reader.get_optimizer_results(optimizer_run_id)`

### Wave 3: Portfolio Builder LiveView
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/portfolio_live.ex`
- `apps/quantedge_web/lib/quantedge_web/live/portfolio_live.html.heex`

**Details:**
Page sections:

1. **Strategy Selection:**
   - Multi-select from saved strategies (checkboxes)
   - Selected strategies appear in allocation panel

2. **Capital Allocation:**
   - Total capital input (number)
   - Per-strategy allocation sliders (percentage, must sum to 100%)
   - Visual bar showing allocation proportions
   - Lock individual allocations, auto-distribute remainder

3. **Run Configuration:**
   - Date range (shared across all strategies)
   - "Run Portfolio Backtest" button → enqueues PortfolioWorker

4. **Portfolio Results (after completion):**
   - Combined equity curve (all strategies + portfolio total)
   - Correlation matrix (NxN grid with color coding: -1 red, 0 white, +1 green)
   - Portfolio-level metrics: Portfolio Sharpe, Diversification Benefit, Max Portfolio DD
   - Per-strategy contribution breakdown (pie chart)

### Wave 4: Correlation Matrix Component
**Files:**
- `apps/quantedge_web/assets/js/hooks/correlation_matrix.js` — Matrix viz hook

**Details:**
- NxN grid where N = number of strategies
- Cell color: diverging colorscale (red -1 to blue +1)
- Cell text: correlation coefficient (2 decimal places)
- Row/column labels: strategy names
- Diagonal always 1.00 (highlighted)
- Tooltip: "Strategy A × Strategy B: 0.35"

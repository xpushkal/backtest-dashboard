# Plan 07-06: Results Viewer — Heatmap, Greeks, Monte Carlo, Walk-Forward

## Objective
Advanced analytics visualizations on the results page: monthly PnL heatmap, Greeks attribution chart, Monte Carlo fan chart, and walk-forward analysis table.

## Requirements Addressed
- UI-05: Monthly PnL heatmap
- UI-06: Greeks attribution chart
- UI-07: Monte Carlo fan chart
- UI-08: Walk-forward results table

## Wave Execution

### Wave 1: Monthly PnL Heatmap
**Files:**
- `apps/quantedge_web/assets/js/hooks/heatmap_chart.js` — Canvas/SVG heatmap hook
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.html.heex` — Add tab

**Details:**
- Grid: rows = months (Jan–Dec), columns = years (2021–2024)
- Cell color: gradient from deep red (max loss) through neutral gray to deep green (max profit)
- Cell text: ₹ PnL value
- Row/column totals on edges
- Data from `Duck.Reader.get_monthly_pnl(run_id)`
- Rendered as SVG via JS hook `HeatmapChart` for pixel-perfect control
- Tooltip on hover: Month, Year, PnL, Trade Count, Win Rate

### Wave 2: Greeks Attribution Chart
**Files:**
- `apps/quantedge_web/assets/js/hooks/greeks_chart.js` — Stacked bar chart hook

**Details:**
- Stacked bar chart (Chart.js): X = time periods (monthly), Y = PnL contribution
- 5 stacks: Delta (blue), Gamma (purple), Theta (green), Vega (orange), Unexplained (gray)
- Shows which Greek drove profits/losses each month
- Data from DuckDB metrics (greeks_pnl_attribution)
- Toggle between monthly and weekly aggregation

### Wave 3: Monte Carlo Fan Chart
**Files:**
- `apps/quantedge_web/assets/js/hooks/monte_carlo_chart.js` — Fan chart hook

**Details:**
- 1000 simulated equity curves (thin gray lines, 5% opacity)
- 5 percentile bands: P5 (red), P25 (orange), P50 (yellow), P75 (light green), P95 (green)
- Actual equity curve overlaid in bold cyan
- X axis: trade number, Y axis: equity
- Data from DuckDB metrics (monte_carlo_results)
- Rendered with Chart.js using custom dataset rendering

### Wave 4: Walk-Forward Table
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.html.heex` — Table component

**Details:**
- Data table with columns: Window #, IS Period, OOS Period, IS Sharpe, OOS Sharpe, Degradation Ratio, IS PnL, OOS PnL
- Color-coded: Degradation < 0.5 = red (overfitting warning), 0.5-0.8 = yellow, > 0.8 = green
- Summary row: Average IS Sharpe, Average OOS Sharpe, Average Degradation
- Verdict badge: "Robust" (avg degradation > 0.7) / "Suspect" / "Overfit"
- Data from DuckDB metrics (walk_forward_results)

## Verification
- [ ] Heatmap renders correct month/year grid with PnL colors
- [ ] Greeks chart shows 5-component stacked bars
- [ ] Monte Carlo renders 1000 curves with percentile bands
- [ ] Walk-forward table shows all windows with degradation coloring

- Monte Carlo fan chart renders in under 2 seconds
- Walk-forward verdict matches manual calculation
- All 4 advanced visualizations accessible via tabs on results page

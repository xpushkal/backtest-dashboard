# Plan 07-05: Results Viewer — Equity Curve, Hero Stats, Metrics

## Objective
Primary results page showing equity curve chart, hero stats cards, trade log, and full metrics table.

## Requirements Addressed
- UI-04: Results viewer with equity curve chart
- UI-09: Hero stats cards (PnL, CAGR, Win Rate, Max DD, Sharpe, PF, Trades, Premium Capture)

## Wave Execution

### Wave 1: Run Show LiveView + Hero Stats
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.ex`
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.html.heex`

**Details:**
Page header: Strategy name, underlying badge, date range, run duration

Hero stats row (8 cards, 4 per row):
1. Total PnL (₹ formatted, green/red based on sign)
2. CAGR (percentage)
3. Win Rate (percentage with bar fill)
4. Max Drawdown (percentage, always red)
5. Sharpe Ratio (number, colored by quality: >2 green, 1-2 yellow, <1 red)
6. Profit Factor (number)
7. Total Trades (count)
8. Premium Capture % (percentage)

Data source: `result_summary` from Postgres (fast) + DuckDB metrics for full detail.

### Wave 2: Equity Curve Chart
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.html.heex` — chart container
- `apps/quantedge_web/assets/js/hooks/equity_chart.js` — Chart.js hook

**Details:**
- Use Chart.js (loaded via CDN or vendor) with a LiveView JS hook
- Dual-axis line chart:
  - Primary Y: Equity value (line, cyan)
  - Secondary Y: Drawdown % (filled area, red with 20% opacity)
- X axis: Date
- Tooltip: Date, Equity, Drawdown%, Daily PnL
- Data loaded from DuckDB via `Duck.Reader.get_equity_curve(run_id)`
- Responsive, fills container width
- PhxHook: `EquityChart` — receives data via `push_event`, renders Chart.js

### Wave 3: Tabs — Trades, Metrics, Details
**Files:**
- `apps/quantedge_web/lib/quantedge_web/live/run_live/show.html.heex`

**Details:**
Tab bar below equity chart with 3 tabs:

**Tab 1: Trade Log**
- Data table with columns: #, Entry Time, Exit Time, Exit Reason, PnL Gross, PnL Net, Bars Held
- PnL columns colored green/red
- Sortable by any column
- Paginated (50 per page) for runs with 1000+ trades
- Data from `Duck.Reader.get_trades(run_id)`

**Tab 2: All Metrics**
- Two-column key-value display of all 75+ metrics
- Grouped by category: Returns, Risk, Trade Stats, Options, Time-Based
- Each metric shows: Name, Value, optional sparkline or bar
- Data from `Duck.Reader.get_metrics(run_id)`

**Tab 3: Run Details**
- Strategy TOML (read-only code block with syntax highlighting)
- Run config: date range, capital, brokerage, slippage
- Timestamps: created, started, completed
- "Re-run" button (clones config, creates new run)
- "Export Results CSV" button

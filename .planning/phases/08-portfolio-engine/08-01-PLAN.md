# Plan 08-01: Portfolio Config (TOML) and PortfolioEngine Orchestrator

## Objective
Implement the `PortfolioConfig` TOML schema and the `PortfolioEngine` orchestrator that runs N strategies concurrently over a shared timeline, respecting individual capital allocation percentages.

## Requirements Addressed
- PORT-01: Multi-strategy portfolio configuration
- PORT-05: Capital allocation percentages respected per strategy

## Wave Execution

### Wave 1: PortfolioConfig TOML Schema
**Files:**
- `apps/quantedge_core/crates/portfolio/src/config.rs`

**Details:**
Define `PortfolioConfig` that parses a portfolio TOML:

```toml
[portfolio]
name = "BN + NF Straddle"
total_capital = 500000

[[strategies]]
name = "BN Short Straddle"
toml_path = "strategies/bn_straddle.toml"    # or inline TOML
allocation_pct = 60.0
underlying = "BANKNIFTY"

[[strategies]]
name = "NF Iron Condor"
toml_path = "strategies/nf_condor.toml"
allocation_pct = 40.0
underlying = "NIFTY"
```

Structs:
- `PortfolioConfig { name, total_capital, strategies: Vec<StrategyAllocation>, date_from, date_to }`
- `StrategyAllocation { name, strategy_config: StrategyConfig, allocation_pct, underlying, allocated_capital }`
- Validation: allocation_pct must sum to 100.0 (±0.1 tolerance)
- `allocated_capital = total_capital * allocation_pct / 100.0`
- Support both inline TOML and file path reference

### Wave 2: PortfolioEngine Orchestrator
**Files:**
- `apps/quantedge_core/crates/portfolio/src/engine.rs`
- `apps/quantedge_core/crates/portfolio/src/lib.rs` — re-export

**Details:**
`PortfolioEngine::run(config: &PortfolioConfig, bars_map: &HashMap<String, Vec<SimBar>>) -> PortfolioResult`

Orchestration flow:
1. For each strategy in config.strategies:
   - Look up bars from `bars_map[&strategy.underlying]`
   - Run `SimRunner::run()` with `strategy.strategy_config` and allocated capital
   - Collect `RunResult` per strategy
2. Merge all per-strategy equity curves into a combined timeline:
   - For each unique date across all strategies:
     - Portfolio equity = sum of all strategy equities on that date
   - Build `combined_equity: Vec<EquityPoint>`
3. Return `PortfolioResult`:

```rust
pub struct PortfolioResult {
    pub strategy_results: Vec<StrategyResult>,
    pub combined_equity: Vec<EquityPoint>,
    pub combined_trades: Vec<ClosedTrade>,  // all trades, tagged with strategy_name
    pub portfolio_metrics: HashMap<String, f64>,
}

pub struct StrategyResult {
    pub name: String,
    pub underlying: String,
    pub allocation_pct: f64,
    pub allocated_capital: f64,
    pub run_result: RunResult,
    pub metrics: HashMap<String, f64>,
}
```

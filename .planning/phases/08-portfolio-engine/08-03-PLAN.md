### Wave 1: Combined Equity Curve Builder
**Files:**
- `apps/quantedge_core/crates/portfolio/src/equity.rs`

**Details:**
Merge N strategy equity curves into a single portfolio equity curve:

```rust
pub struct CombinedEquity {
    pub dates: Vec<NaiveDate>,
    pub portfolio_equity: Vec<f64>,        // sum of all strategy equities
    pub strategy_equities: Vec<Vec<f64>>,  // per-strategy equity on each date
    pub strategy_names: Vec<String>,
    pub daily_returns: Vec<f64>,           // portfolio daily returns
}

impl CombinedEquity {
    pub fn build(
        strategy_results: &[StrategyResult],
        total_capital: f64,
    ) -> Self;
}
```

Algorithm:
1. Collect all unique dates from all strategy equity curves
2. Sort dates chronologically
3. For each date: interpolate each strategy's equity (forward-fill if date missing for a strategy)
4. `portfolio_equity[i] = sum(strategy_equities[j][i] for j in strategies)`
5. Compute `daily_returns[i] = (portfolio_equity[i] / portfolio_equity[i-1]) - 1.0`

### Wave 2: CorrelationMatrix
**Files:**
- `apps/quantedge_core/crates/portfolio/src/correlation.rs`

**Details:**
NxN correlation matrix between strategy daily returns:

```rust
pub struct CorrelationMatrix {
    pub strategy_names: Vec<String>,
    pub matrix: ndarray::Array2<f64>,  // NxN Pearson correlation
}

impl CorrelationMatrix {
    /// Compute Pearson correlation between each pair of strategy daily PnL series.
    pub fn compute(
        strategy_daily_pnls: &[Vec<f64>],
        strategy_names: &[String],
    ) -> Self;

    /// Get correlation between two strategies by name.
    pub fn get(&self, a: &str, b: &str) -> Option<f64>;

    /// Average pairwise correlation (excluding diagonal).
    pub fn avg_correlation(&self) -> f64;

    /// Serialize to JSON-compatible structure: Vec<Vec<f64>>
    pub fn to_json(&self) -> serde_json::Value;
}
```

Pearson correlation formula:
- `corr(X, Y) = cov(X,Y) / (std(X) * std(Y))`
- Use `ndarray` for matrix operations

### Wave 3: Portfolio Metrics (MET-05)
**Files:**
- `apps/quantedge_core/crates/portfolio/src/metrics.rs`

**Details:**
Compute the 7 portfolio-level metrics from MET-05:

```rust
pub struct PortfolioMetrics {
    pub portfolio_sharpe: f64,
    pub portfolio_sortino: f64,
    pub portfolio_max_drawdown_pct: f64,
    pub portfolio_total_pnl: f64,
    pub peak_margin_used: f64,
    pub capital_efficiency: f64,        // total_pnl / peak_margin
    pub avg_concurrent_trades: f64,     // avg number of open positions at any time
    pub diversification_benefit: f64,   // 1 - (portfolio_vol / sum_of_strategy_vols)
    pub avg_correlation: f64,           // from correlation matrix
}

impl PortfolioMetrics {
    pub fn compute(
        combined_equity: &CombinedEquity,
        strategy_results: &[StrategyResult],
        correlation: &CorrelationMatrix,
        margin_tracker: &PortfolioMarginTracker,
        total_capital: f64,
    ) -> Self;
}
```

Key formulas:
- **Diversification Benefit**: `1 - (σ_portfolio / Σ(w_i × σ_i))` where σ = annualized volatility, w_i = allocation weight
- **Portfolio Sharpe**: `(portfolio_annualized_return - 0.065) / portfolio_annualized_volatility` (6.5% risk-free for India)
- **Capital Efficiency**: `total_pnl / peak_margin_used`
- **Avg Concurrent Trades**: sum of (bars where position open per strategy) / total bars

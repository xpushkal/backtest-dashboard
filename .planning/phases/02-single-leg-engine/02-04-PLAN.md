## Objective

Compute the 20 core metrics from a `RunResult` (trades + snapshots). These metrics
are the foundation — Phase 5 extends to 75+. The metrics engine takes a Vec<ClosedTrade>
and Vec<PositionSnapshot> and produces a MetricsResult struct.

## must_haves

- MetricsEngine::compute(trades, snapshots, config) → MetricsResult
- All 20 core metrics computed correctly:
  - Return: total_pnl_net, total_pnl_gross, cagr, roi_pct, expectancy, profit_factor, win_rate_pct, avg_win, avg_loss, largest_win, largest_loss
  - Risk: max_drawdown_inr, max_drawdown_pct, sharpe_ratio
  - Trade: total_trades, avg_hold_bars, sl_hit_rate_pct, target_hit_rate_pct, time_exit_rate_pct, total_brokerage
- Sharpe ratio uses 6.5% annual risk-free rate (India)
- Edge cases: 0 trades → metrics default to 0, no division by zero

use chrono::NaiveDate;

#[derive(Debug, Clone, Default)]
pub struct MetricsResult {
    // Return metrics (11)
    pub total_pnl_gross: f64,
    pub total_pnl_net: f64,
    pub cagr: f64,                   // compound annual growth rate
    pub roi_pct: f64,                // total return / capital * 100
    pub expectancy: f64,             // avg PnL per trade
    pub profit_factor: f64,          // gross_profit / gross_loss (abs)
    pub win_rate_pct: f64,
    pub avg_win: f64,
    pub avg_loss: f64,
    pub largest_win: f64,
    pub largest_loss: f64,

    // Risk metrics (3)
    pub max_drawdown_inr: f64,       // peak-to-trough in INR
    pub max_drawdown_pct: f64,       // peak-to-trough as % of peak equity
    pub sharpe_ratio: f64,           // (annualized return - Rf) / annualized vol

    // Trade analytics (6)
    pub total_trades: u32,
    pub avg_hold_bars: f64,
    pub sl_hit_rate_pct: f64,
    pub target_hit_rate_pct: f64,
    pub time_exit_rate_pct: f64,
    pub total_brokerage: f64,
}

pub struct MetricsEngine;

impl MetricsEngine {
    pub fn compute(
        trades: &[ClosedTrade],
        snapshots: &[PositionSnapshot],
        capital: f64,
        start_date: NaiveDate,
        end_date: NaiveDate,
    ) -> MetricsResult {
        if trades.is_empty() {
            return MetricsResult::default();
        }

        let mut result = MetricsResult::default();

        // Return metrics
        result.total_trades = trades.len() as u32;
        result.total_pnl_gross = trades.iter().map(|t| t.pnl_gross).sum();
        result.total_pnl_net = trades.iter().map(|t| t.pnl_net).sum();
        result.total_brokerage = trades.iter().map(|t| t.brokerage + t.stt).sum();
        result.roi_pct = (result.total_pnl_net / capital) * 100.0;

        let wins: Vec<f64> = trades.iter().filter(|t| t.pnl_net > 0.0).map(|t| t.pnl_net).collect();
        let losses: Vec<f64> = trades.iter().filter(|t| t.pnl_net <= 0.0).map(|t| t.pnl_net).collect();

        result.win_rate_pct = (wins.len() as f64 / trades.len() as f64) * 100.0;
        result.avg_win = if wins.is_empty() { 0.0 } else { wins.iter().sum::<f64>() / wins.len() as f64 };
        result.avg_loss = if losses.is_empty() { 0.0 } else { losses.iter().sum::<f64>() / losses.len() as f64 };
        result.largest_win = wins.iter().copied().fold(0.0_f64, f64::max);
        result.largest_loss = losses.iter().copied().fold(0.0_f64, f64::min);
        result.expectancy = result.total_pnl_net / trades.len() as f64;

        let gross_profit: f64 = wins.iter().sum();
        let gross_loss: f64 = losses.iter().map(|l| l.abs()).sum();
        result.profit_factor = if gross_loss > 0.0 { gross_profit / gross_loss } else { f64::INFINITY };

        // CAGR
        let years = (end_date - start_date).num_days() as f64 / 365.25;
        let final_equity = capital + result.total_pnl_net;
        result.cagr = if years > 0.0 && final_equity > 0.0 {
            (final_equity / capital).powf(1.0 / years) - 1.0
        } else { 0.0 };

        // ... drawdown from snapshots, sharpe from daily returns, etc.

        result
    }
}
```

Key formulas:
- CAGR = (final_equity / initial_capital)^(1/years) - 1
- Sharpe = (annualized_return - 0.065) / annualized_volatility
- Max DD: track peak equity from snapshots, DD = (peak - current) / peak
- Annualized vol = daily_std * sqrt(252)
- Daily returns from equity snapshots (end-of-day equity changes)

fn compute_drawdown(snapshots: &[PositionSnapshot]) -> (f64, f64) {
    // Returns (max_drawdown_inr, max_drawdown_pct)
    let mut peak = f64::MIN;
    let mut max_dd_inr = 0.0_f64;
    let mut max_dd_pct = 0.0_f64;

    for snap in snapshots {
        if snap.equity > peak {
            peak = snap.equity;
        }
        let dd = peak - snap.equity;
        if dd > max_dd_inr {
            max_dd_inr = dd;
            max_dd_pct = dd / peak * 100.0;
        }
    }
    (max_dd_inr, max_dd_pct)
}

fn compute_sharpe(snapshots: &[PositionSnapshot], capital: f64) -> f64 {
    // Daily returns from end-of-day equity snapshots
    // Group snapshots by date, take last snapshot per day
    let rf_annual = 0.065; // 6.5% India risk-free rate
    let rf_daily = rf_annual / 252.0;

    let daily_equities: Vec<f64> = /* last equity per unique date */;
    let daily_returns: Vec<f64> = daily_equities.windows(2)
        .map(|w| (w[1] - w[0]) / w[0])
        .collect();

    if daily_returns.is_empty() { return 0.0; }

    let mean = daily_returns.iter().sum::<f64>() / daily_returns.len() as f64;
    let variance = daily_returns.iter().map(|r| (r - mean).powi(2)).sum::<f64>()
        / daily_returns.len() as f64;
    let std = variance.sqrt();

    if std == 0.0 { return 0.0; }
    ((mean - rf_daily) / std) * (252.0_f64).sqrt()
}
```

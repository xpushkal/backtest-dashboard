## Objective

Implement the core trading primitives: `Leg` (a single option position with SL
tracking), `Position` (a collection of legs opened together), and `SlStateMachine`
(Active → TrailActivated → Triggered). These are the building blocks the simulation
runner will orchestrate.

## must_haves

- Leg struct with entry price, current price, mark-to-market PnL, lots, direction
- SlStateMachine with Active / TrailActivated / Triggered states
- Position struct holding one leg (Phase 2), capturing entry bar, tracking unrealized PnL
- PositionSnapshot for equity curve capture (bar-level equity point)
- ClosedTrade struct with entry/exit prices, PnL gross/net, brokerage, exit reason

```rust
use crate::config::ExitReason;
use crate::leg::Leg;
use chrono::{NaiveDate, NaiveTime};
use quantedge_data::Bar;

pub struct Position {
    pub legs: Vec<Leg>,            // Phase 2: single leg; Phase 3: multi-leg
    pub entry_date: NaiveDate,
    pub entry_time: NaiveTime,
    pub entry_bar_index: usize,
    pub total_brokerage: f64,      // entry-side brokerage
}

impl Position {
    pub fn new(legs: Vec<Leg>, bar: &Bar, bar_index: usize, brokerage: f64) -> Self { ... }

    pub fn update(&mut self, bar: &Bar) {
        for leg in &mut self.legs {
            leg.update(bar);
        }
    }

    pub fn total_unrealized_pnl(&self) -> f64 {
        self.legs.iter().map(|l| l.unrealized_pnl).sum()
    }

    pub fn total_bars_held(&self) -> u32 {
        self.legs.iter().map(|l| l.bars_held).max().unwrap_or(0)
    }
}

pub struct ClosedTrade {
    pub entry_date: NaiveDate,
    pub entry_time: NaiveTime,
    pub exit_date: NaiveDate,
    pub exit_time: NaiveTime,
    pub entry_price: f64,          // leg entry premium
    pub exit_price: f64,           // leg exit premium
    pub entry_spot: f64,
    pub exit_spot: f64,
    pub option_type: OptionType,
    pub position_side: PositionSide,
    pub lots: u32,
    pub lot_size: u32,
    pub pnl_gross: f64,            // before brokerage/STT
    pub brokerage: f64,            // entry + exit brokerage
    pub stt: f64,                  // STT on sell side only
    pub slippage_cost: f64,
    pub pnl_net: f64,              // pnl_gross - brokerage - stt - slippage
    pub exit_reason: ExitReason,
    pub bars_held: u32,
}

pub struct PositionSnapshot {
    pub date: NaiveDate,
    pub time: NaiveTime,
    pub spot: f64,
    pub equity: f64,               // capital + cumulative realized PnL + unrealized PnL
    pub unrealized_pnl: f64,
    pub cumulative_pnl: f64,       // sum of all closed trade PnL net
}
```

STT calculation for sell side: `0.0625% of (exit_price * quantity)` (current FNO STT rate).
Brokerage: `brokerage_per_lot * lots * 2` (entry + exit, both sides).

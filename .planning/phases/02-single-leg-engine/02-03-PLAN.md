## Objective

Implement the core simulation loop that iterates over bars, opens/closes positions,
and the execution engine that applies slippage, brokerage, and STT. Also implement
ATM strike selection. This is the heart of the backtester.

## must_haves

- SimRunner::run() that takes StrategyConfig + bars → RunResult
- Entry logic: open position at entry_time on ATM strike
- Exit logic: priority order — SL → Target → TimeExit → EndOfData
- ExecutionEngine: slippage (fixed_pts, percent), brokerage (per-lot), STT (0.0625% sell)
- StrikeSelector: find ATM bar from bar slice at given timestamp
- RunResult struct with Vec<ClosedTrade> and Vec<PositionSnapshot>
- No position overlap (max_concurrent_trades = 1 for Phase 2)

```rust
use quantedge_data::Bar;
use crate::config::{LegConfig, StrikeMode, OptionType};

pub struct StrikeSelector;

impl StrikeSelector {
    /// Find the bar matching the desired strike for a leg config.
    /// For ATM offset mode: find bar where strike_offset == leg.strike_offset
    /// and option_type matches.
    ///
    /// Returns None if no matching bar found (no data for that strike).
    pub fn select_bar<'a>(
        leg: &LegConfig,
        bars_at_time: &'a [Bar],
    ) -> Option<&'a Bar> {
        match leg.strike_mode {
            StrikeMode::AtmOffset => {
                let target_type = match leg.option_type {
                    OptionType::CE => "CE",
                    OptionType::PE => "PE",
                };
                bars_at_time.iter().find(|b| {
                    b.strike_offset == leg.strike_offset
                        && b.option_type == target_type
                })
            }
            // Delta, Premium, PercentOtm stubbed for Phase 3
            _ => None,
        }
    }
}
```

```rust
use crate::config::{PositionSide, SlippageModel, StrategyConfig};

pub struct ExecutionEngine;

impl ExecutionEngine {
    /// Apply slippage to a fill price.
    /// For sells: entry price reduced (worse fill = lower premium received).
    /// For buys: entry price increased (worse fill = higher premium paid).
    pub fn apply_slippage(
        price: f64,
        side: PositionSide,
        model: &SlippageModel,
        value: f64,
    ) -> f64 {
        match model {
            SlippageModel::None => price,
            SlippageModel::FixedPts => match side {
                PositionSide::Sell => (price - value).max(0.01),
                PositionSide::Buy => price + value,
            },
            SlippageModel::Percent => match side {
                PositionSide::Sell => price * (1.0 - value / 100.0),
                PositionSide::Buy => price * (1.0 + value / 100.0),
            },
            SlippageModel::VolumeBased => price, // stub for now
        }
    }

    /// Calculate brokerage for a trade (entry + exit).
    /// brokerage_per_lot * lots * 2 (both sides)
    pub fn calculate_brokerage(brokerage_per_lot: f64, lots: u32) -> f64 {
        brokerage_per_lot * lots as f64 * 2.0
    }

    /// Calculate STT (Securities Transaction Tax).
    /// FNO sell side: 0.0625% of (premium * quantity)
    /// Only on sell-side transactions.
    pub fn calculate_stt(
        exit_price: f64,
        quantity: u32, // lots * lot_size
        stt_on_sell: bool,
        position_side: PositionSide,
    ) -> f64 {
        if !stt_on_sell {
            return 0.0;
        }
        // STT applies on sell side of futures/options
        // For short positions: STT on entry (sell) and exit if buy-back
        // Simplified: apply 0.0625% on sell value
        let sell_value = exit_price * quantity as f64;
        sell_value * 0.000625
    }

    /// Calculate total slippage cost for entry + exit.
    pub fn calculate_slippage_cost(
        entry_price: f64,
        exit_price: f64,
        model: &SlippageModel,
        value: f64,
        lots: u32,
        lot_size: u32,
    ) -> f64 {
        let quantity = (lots * lot_size) as f64;
        match model {
            SlippageModel::None => 0.0,
            SlippageModel::FixedPts => value * quantity * 2.0, // entry + exit
            SlippageModel::Percent => {
                let entry_slip = entry_price * (value / 100.0) * quantity;
                let exit_slip = exit_price * (value / 100.0) * quantity;
                entry_slip + exit_slip
            }
            SlippageModel::VolumeBased => 0.0,
        }
    }
}
```

```rust
pub struct RunResult {
    pub trades: Vec<ClosedTrade>,
    pub snapshots: Vec<PositionSnapshot>,
    pub total_bars: usize,
}

pub struct SimRunner;

impl SimRunner {
    pub fn run(config: &StrategyConfig, bars: &[Bar], lot_size: u32) -> RunResult {
        let mut position: Option<Position> = None;
        let mut trades: Vec<ClosedTrade> = Vec::new();
        let mut snapshots: Vec<PositionSnapshot> = Vec::with_capacity(bars.len() / 20);
        let mut cumulative_pnl: f64 = 0.0;

        // Group bars by (date, time) for per-timestamp processing
        // Build index: HashMap<(NaiveDate, NaiveTime), &[Bar]>

        for (date, time, bars_at_time) in grouped_bars(bars) {
            // 1. ENTRY: if no position and time == entry_time
            if position.is_none() && time == config.entry_time {
                if let Some(pos) = Self::try_open(config, &bars_at_time, lot_size, snapshot_idx) {
                    position = Some(pos);
                }
            }

            // 2. UPDATE: mark-to-market all legs
            if let Some(ref mut pos) = position {
                pos.update_from_bars(&bars_at_time);

                // 3. EXIT CHECKS (priority order)
                let exit = Self::check_exits(config, pos, time);
                if let Some(reason) = exit {
                    let trade = Self::close_position(pos, &bars_at_time, reason, config, lot_size);
                    cumulative_pnl += trade.pnl_net;
                    trades.push(trade);
                    position = None;
                }
            }

            // 4. SNAPSHOT for equity curve
            let unrealized = position.as_ref().map_or(0.0, |p| p.total_unrealized_pnl());
            snapshots.push(PositionSnapshot {
                date,
                time,
                spot: bars_at_time.first().map_or(0.0, |b| b.spot),
                equity: config.capital + cumulative_pnl + unrealized,
                unrealized_pnl: unrealized,
                cumulative_pnl,
            });
        }

        // Close any open position at end of data
        if let Some(pos) = position {
            // ... close with EndOfData reason
        }

        RunResult { trades, snapshots, total_bars: bars.len() }
    }

    fn check_exits(config: &StrategyConfig, pos: &Position, time: NaiveTime) -> Option<ExitReason> {
        // Priority: per-leg SL → per-leg target → time exit
        for leg in &pos.legs {
            if let Some(r) = leg.check_sl() { return Some(r); }
        }
        for leg in &pos.legs {
            if let Some(r) = leg.check_target() { return Some(r); }
        }
        if time >= config.exit_time {
            return Some(ExitReason::TimeExit);
        }
        None
    }
}
```

The runner processes bars grouped by unique (date, time) tuples in chronological order.
At each timestamp it:
1. Checks if it's entry time and no position open → opens
2. Updates all leg MTM prices
3. Checks exit conditions in priority order
4. Captures equity snapshot

Performance note: avoid cloning bars. Use references and indices.
Snapshot frequency: once per unique (date, time) — typically 375 per day × ~1000 days ≈ 375K snapshots.
